FDNS Search Launcher Homepage

Reminders: Follow all Standard Operating Procedures (SOPs) and guidance set by HQ FDNS Social Media Division while using this tool. All required platform searches for each selector must be documented for complete SM check. Follow proper referral process if needed depending on guidance and platform behaviors. This tool is only to be used for conducting social media checks with no exception for any personal browsing. R2D2 logs user web traffic history and may be audited. Officers will have access revoked without warning for violations of the Rules of Behavior.

Confirmed Account Tools:


These are links to find unique IDs related to a Subject's confirmed account on different social media platforms.
  • Facebook: Lookup-ID OR Find My FB ID OR:
    • Individual accounts: find "fb://profile/" in the source code.
    • Business or Group accounts: find "fb://page/" in the source code.
  • Twitter: Find Twitter ID OR: find "profile_user" (in Firefox with browser add-on enabled) in the source code.
  • Instagram: Find Instagram User ID OR:
    • For ImgInn: find "data-id=" in the source code.
    • For PicUki: find "==&id=" in the source code.
    • For GramHo: the account ID is included in the URL after the account screenname.
    • From an account: find "ProfilePage_" in the source code
  • YouTube: YouTube Channel ID OR: find "externalID" in the source code.

Additional Tools:


UI Masker to view Twitter: SMD has found that using User Interface masking add-ons may permit you to see additional public information on platforms. Users will be able to change settings this through the add-on icon to the right of the R2D2 URL bar. These settings will take effect on any newly opened or refreshed tabs.

IG 3rd Party Sites: SMD has found that these websites have begun working once again after changes to the API for Instagram in December 2020. In the event you try all 3rd party IG sites and none work, please refer to the ECN for current procedures.
