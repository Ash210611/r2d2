Phone Number Search:

This field searches the following sites: Google.

Platform ID Tools:

These are links to find unique account ID's on different social media platforms.
  • Find Facebook ID: Lookup-ID.com OR Find My Facebook ID OR: find "profile_owner" in the source code.
  • Find Business or Group Facebook ID by searching "entity_id" in the source code.
  • Find Twitter ID: Get A User's Twitter ID OR: find "profile_user" (in Firefox) or "data-user-id" (in Chrome) in the source code.
  • Find Instagram ID: Find IG User ID OR: find "profilepage_" in the source code.
  • Find YouTube ID: YouTube Channel ID OR: find "externalID" in the source code.

UPDATES TO THE SEARCH LAUNCHER:

06/09/20:

  • Users will be able to select their egress (location) after logging into R2D2. It will launch a new instance of the browser window and will NOT carry over any open windows when a new egress location is chosen. Users are advised to change egress points periodically to spread out data traffic.
  • Javascript in Chrome browser is blocked by default, as a way to prevent platforms from blocking publicly available results.
  • Twitter will show a "legacy view" in Chrome due to Javascript being disabled. Firefox is currently unable to display full searches in Twitter.
  • Consult the Social Media ECN for any current advisories related to logged-in account referral process.
  • Alternate View button removed from name search bar.
  • A Twitter "Directory" Tab button has been added for use by Firefox browser.
  • Additional Instagram and new platform links in custom search bar.
  • Added ID source code for Facebook accounts that belong to a business/organization. Additional link for Facebook also included.
  • Due to advisories related to Govt. use of TikTok on devices, the platform has been removed from Search Launcher.
  • Backend improvements for egress points, including additional egress points for users.

12/08/19:

  • Backend improvements in the formation of instances, including removal of persistent cookies, to help anonymize web traffic. Remember, you can always clear your web traffic by pressing CTRL+SHIFT+DELETE and clearing all web history.
  • Pop-ups should be automatically enabled for the search launcher for users. If not, remember to enable them by clicking on the pop-up in Firefox or the red icon in the URL bar in Chrome.
  • Tabs that were previously opened upon launch are now saved as links in the "Bookmarks" folder. In order to launch all bookmarks, right click on the folder and select "open all".
