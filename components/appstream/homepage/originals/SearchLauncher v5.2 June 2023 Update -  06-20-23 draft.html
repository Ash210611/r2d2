Confirmed Account Tools:


These are links and terms to find unique IDs related to a Subject's confirmed account on different social media platforms. All confirmed accounts must have their unique User ID recorded (with or without any found information of interest).
  • Facebook: Lookup-ID OR Find My FB ID OR search the source code for:
    • Logged-in FB account: UserID immediately after the name used on account.
    • Business listings and group pages: pageID

  • Instagram: Find Instagram User ID OR search the source code for:
    • Logged-in IG account: ProfilePage_
    • PicNob: "userid" value=
    • Picuki: let id =
    • Dumpoir does not display account ID # in the source code.

  • Twitter: Find Twitter ID OR TweeterID
    • Twitter does not currently display User IDs in the source code.

  • YouTube: YouTube Channel ID OR: find externalID in the source code.

  • TikTok: Find TikTok User ID OR: find "authorID" in the source code.

Additional Tools/Tips/Guidance:


Additional information about known platform behavior and other common occurences while reviewing open-source material.
Logged-In Checks: All five core sites must be checked for a complete social media check. Facebook and Twitter require the use of logged-in accounts to review public information.
  • Facebook: If you are not an Assistor with a FB account, a logged-in referral to an Assistor is required. Assistors must run names, aliases, email handles, and usernames.
  • Twitter: If you are not an Assistor with a TW account, a logged-in referral to an Assistor is required. Assistors must run names, aliases, email handles, and usernames.
  • Instagram: Searches on IG (third-party IG or IG.com) are required for all social media checks. If there is information of interest found on an IG account, that information must be captured on Instagram.com by an Assistor.
  • LinkedIn: Referrals for logged-in account review should only be used when the social media check is directly related to information about employment or work histories. LinkedIn does not display information about marriages or relationship status.
Search Launcher & Instagram.com: The Search Launcher has added an "Open Instagram Tab" link in the blue box above. This cannot directly search instagram.com due to the architecture of the platform. Use of approved third-party instagram sites suffice for a search of IG for accounts. If an Officer is unable to use a third-party IG site or prefers to search Instagram.com directly, they would use the dynamic search bar on that platform. Remember to record all information of interest on the original instagram.com account. (Not a requirement if only a confirmed accounts with no Information of Interest is found.)

Pop-ups when Reviewing Twitter: SMD has found that a "sign-in" pop up window currently halts a full review of accounts. Closing the window (click "sign in", then immediately closing the window using the "X" button) will permit Conductors to see the remaining information. As always, privacy guidelines prevent any unauthorized signing into platform accounts.

Narrowing Information: Adding city of birth or city of residence to Facebook (in the "City" field) and Google can narrow results for common names or searches with many results. It may be important to include country calling codes and phone number formatting from the country of origin for phone selectors. Additionally, adding country of origin + phone number may clarify results in Google. (ex. Honduras = +504 5555-5555).

DocSumo: OCR translation tool DocSumo has been added as a default add-on in Chrome. Currently, DocSumo does not have an equivalent tool inside Firefox. The Online OCR Scanner file upload should work in either browser.

Please share any additional tips you find with SMD!
